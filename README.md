# Equivalent Fractions - iFrações Module

This project is an educational game focused on teaching equivalent fractions. It was developed as an extension for the [iFrações](https://www.usp.br/line/ifractions/) ecosystem , a project by the Laboratory of Informatics in Education ([LINE](https://line.ime.usp.br)) at Institute of Mathematics, Statistics and Computer Science of the University of São Paulo (IME-USP).

## Educational Objective
The game aims to help primary and secondary school students overcome the recurring difficulty of understanding that different fractional representations can express the same quantity. Through a visual and interactive interface, students will:
- Identify equivalent fractions using colored proportional bars;
- Understand fractions as parts of a whole.
- Relate abstract concepts to visual representations inspired by the Rainbow Fraction and Cuisenaire systems. 

## Theme and Mechanics
The player takes on the role of an apprentice engineer responsible for constructing a building. The central challenge involves constructing walls by combining blocks of various sizes; for instance, a player might use two $1/4$ blocks or four $1/8$ blocks to fill a required $1/2$ space.The system provides immediate visual feedback after each action, allowing the student to see if the combination matches the target size and highlighting any conceptual errors. The building consists of 4 levels (floors), with a progressive increase in the difficulty of the denominators used.

This module is an open-source project developed using standard web technologies to ensure portability and easy integration:
- HTML5: Interface structure;
- CSS3: Styling and layout;
- JavaScript (Vanilla): Game logic, drag-and-drop mechanics, and equivalence calculations.
